Readme for the data subdirectory

subsample.shared - subsampled shared file for Jan400 analysis
groups.summary - has alpha diversity data
an.cons.taxonomy - taxonomy file
simplytaxonomy - generated using the trimTax.py script and the an.cons.taxonomy file
.dist - distance matrices for adonis analysis

/mothur - mothur files
	/raw - includes the Jan400.files file to run make.contigs in mothur
/archive_dist_files - dist files used for earlier analysis that arent needed to render final pdf
